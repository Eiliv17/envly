Envly is a desktop application for managing secrets and environment variables. It provides encrypted vaults where you store all your sensitive data, then lets you organize secrets across multiple projects and environments and inject them into your local development workflow safely.

## The Problem

Developers juggle `.env` files across dozens of repositories. Secrets get copy-pasted between machines, accidentally committed to version control, and fall out of sync between staging and production. There's no single place to see all your secrets, rotate them, or know which project depends on what.

## How Envly Solves It

Envly gives you a centralized, encrypted store for all your secrets with a simple workflow:

1. **Store secrets once** in an encrypted vault with strong encryption (XChaCha20-Poly1305).
2. **Organize by project** — each project maps to a local repository on your machine.
3. **Define environments** (development, staging, production) and map local `.env` keys to your stored secrets.
4. **Activate an environment** — Envly writes a secure temp file and creates a symlink in your project directory. Your dev tools read the `.env` file as usual.
5. **Stay safe** — if the symlink is accidentally committed to git, it's just a broken link pointing to a nonexistent path. No secrets are leaked.

## Key Features

- **Encrypted vaults** with passphrase or OS keychain authentication
- **Multiple vaults** for different contexts (work, personal, client projects)
- **Per-project, per-environment** secret mappings
- **Symlink-based injection** that never writes plaintext secrets into your source tree
- **Expiration tracking** with UI warnings to encourage secret rotation
- **Bulk import** from existing `.env` files
- **Cross-platform** support for macOS, Windows, and Linux
- **Open source** under the Apache 2.0 license

## Tech Stack

Envly is built with:

- **Tauri 2** — lightweight desktop application framework
- **Rust** — backend handling encryption, file operations, and symlink management
- **Nuxt 4 + Vue 3** — frontend UI with the `@nuxt/ui` component library
- **Tailwind CSS** — styling
