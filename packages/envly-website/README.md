# Envly Website

The marketing site and documentation for [Envly](https://github.com/eiliv17/envly), built with [Nuxt](https://nuxt.com), [Nuxt UI](https://ui.nuxt.com), and [Nuxt Content](https://content.nuxt.com).

## Setup

Make sure to install dependencies from the repository root:

```bash
pnpm install
```

## Development

Start the development server on `http://localhost:3000`:

```bash
pnpm dev:website
```

## Production

Build the application for production:

```bash
pnpm build:website
```
